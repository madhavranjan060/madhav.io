## How To Run

1. Download or copy all files.
2. Keep the folder structure unchanged.
3. Open **`index.html`** in any browser.


## How To Customize

### Change Name

To personalize the website with your own name, you need to update it in two files:

1. **HTML File (`index.html`)**:
   * **Tab Title**: Look for `<title>Madhav Ranjan | Portfolio</title>` near the top and change it.
   * **Navbar Logo**: Look for `<h2>Madhav</h2>` inside the `<nav>` tag and swap it with your name.
   * **Footer Copyright**: Scroll to the bottom and update `© 2026 Madhav Ranjan` inside the `<footer>` tag.

2. **JavaScript File (`assets/js/script.js`)**:
   * **Typing Animation**: Change the text on the very first line: `const text = "Madhav Ranjan";` to your own name.


### Change Social Links

Find this block of code inside **`index.html`**:

```html
<div class="socials">

<a href="https://instagram.com_">
Instagram
</a>

<a href="#">
Telegram
</a>

<a href="#">
Twitter/X
</a>

</div>
```

Change the values inside the `href="..."` attributes and replace links with your own personal profile URLs.


### Add Images

Put images inside the **`assets/images/`** directory and connect them in HTML using this format:

```html
<img src="assets/images/your-image-name.jpg" alt="Description">
```
